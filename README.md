# Vibha_garden_sensor

## BASE STATION
| NAME     | Pin |
|----------|-----|
| SDA      | A4  |
| SCL      | A5  |
| SOIL     | A7  |
| OK       | D5  |
| LEFT     | D4  |
| RIGHT    | D7  |
| BACK     | D6  |
| SOUND    | D3  |
| DHT11    | D2  |
| TX       | D0  |
| RX       | D1  |
| SCK      | D13 |


### MODULES USED
- SOUND (LM393)
- DHT11
- MPU6050 GYROSCOPE
- SOIL MOISTURE
- ESP8266 NODEMCU
- ARDUINO NANO
- NAVIGATION MODULE
- DISPLAY I2C 16*2
